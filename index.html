_.chunk(array, [size=1])

Создает массив элементов, разбитых на группы длиной size. Если array нельзя разделить поровну, последним фрагментом будут оставшиеся элементы.

_.findIndex(array, [predicate=_.identity], [fromIndex=0])

Этот метод подобен _.find, за исключением того, что он возвращает индекс первого элемента, предикат возвращает true вместо самого элемента.

_.flatten(array)

Выравнивает массив на один уровень в глубину.

_.flattenDeep(array)

Рекурсивно сглаживает массив.
